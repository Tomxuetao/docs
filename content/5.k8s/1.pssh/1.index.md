# ParallelSSH

[`parallel-ssh`](https://code.google.com/archive/p/parallel-ssh/source/default/source) 包括 pssh、pscp、prsync、pslurp 和
pnuke工具，其源代码使用 Python语言编写开发的。

## 安装

::code-group
  ```bash [Mac]
  brew install pssh
  ```
  ```bash [CentOS]
  yum install pssh
  ```
  ```bash [Ubuntu]
  apt install pssh
  ```
  ```bash [PIP安装]
  pip insall pssh
  ```
::
